# RAM Schedule Map – Simple Overview

## What This Tool Is

The **RAM Schedule Map** is the tool on the RAM website that shows **upcoming clinics**.

It allows visitors to:

- See where clinics will be held
- Browse clinics on a **map**
- View clinics in a **calendar**
- Filter clinics by state, services, or clinic type
- Click a clinic to learn more

This helps patients and volunteers easily find clinics near them.

---

# Where the Information Comes From

All clinic information starts in the RAM website’s event system called **Modern Events Calendar (MEC)**.

When staff create or update a clinic in MEC:

1. The clinic information is saved in the website.
2. That information is automatically synced to a data file.
3. The Schedule Map reads that file and displays the clinics on the website.

Because of this process, the **Schedule Map automatically stays up to date** whenever clinic information is updated.

---

# What the Map Shows

The Schedule Map allows users to explore clinics in several ways:

### Map View
Users can see clinics displayed on a map and click them for details.

### Calendar View
Users can view clinics organized by date.

### Clinic List
A list of clinics appears beside the map so users can browse upcoming events.

---

# Filters Users Can Apply

Visitors can narrow down clinics by:

- **State**
- **Month**
- **Clinic Type** (popup clinic or telehealth)
- **Services Available**
  - Medical
  - Dental
  - Vision
  - Dentures

This helps users quickly find the clinics that meet their needs.

---

# What We Measure

We also track how people use the Schedule Map so RAM can understand what visitors are looking for.

Examples of what we measure:

- How many people open the schedule map
- Which clinics people click on
- Which states people search for
- Which services people filter by
- How often users click “Learn More” about a clinic

This information helps RAM improve how clinic information is presented on the website.

---

# Reporting Dashboard

The data collected from the Schedule Map is shown in a **Looker Studio dashboard**.

This dashboard allows staff to see:

- Total number of users viewing the schedule
- The most popular clinics
- Which states receive the most interest
- Which services people search for most
- Overall engagement with the clinic schedule

This helps RAM understand how visitors interact with upcoming clinic information.

---

# Why This System Is Helpful

The Schedule Map helps RAM by:

- Making clinic information **easy for visitors to find**
- Keeping clinic data **automatically updated**
- Helping staff understand **what clinics people are most interested in**
- Providing data that supports **planning and reporting**

The RAM Schedule Map:

- Helps visitors find clinics quickly
